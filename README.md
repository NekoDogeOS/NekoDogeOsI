### 技术准备

- 硬件部分
    使用QEMU模拟一台x86计算机。由于自己是在服务器上开发这个操作系统的，如果您
使用的时候想要看到图形界面。请记住不要使用``make qemu-noneDisplay``命令，而请使用
``make qemu``命令获得更好体验。
